Create `backend/src/flywheel/engines/voice_context_writer.py` with two async functions:

**`write_voice_to_context(db, tenant_id, user_id, profile_dict, samples_analyzed)`:**
1. Soft-delete any existing sender-voice entry for this tenant+user: `UPDATE context_entries SET deleted_at=now() WHERE tenant_id=X AND file_name='sender-voice' AND source='email-voice-engine' AND deleted_at IS NULL`
2. Format all 10 voice fields as human-readable markdown content using `_format_voice_content(profile_dict, samples_analyzed)`. Format example:
   ```
   Writing Voice Profile (extracted from 47 sent emails)

   Tone: professional and direct
   Formality: conversational
   Greeting style: Hi {name},
   Sign-off: Best,
   Average length: ~80 words, ~3 sentences
   Paragraph pattern: short single-line
   Question style: direct
   Emoji usage: never
   Characteristic phrases: "happy to help", "let me know", "sounds good"
   ```
3. Insert new ContextEntry with:
   - `tenant_id`, `user_id` from params
   - `file_name="sender-voice"`
   - `source="email-voice-engine"`
   - `detail="Voice profile snapshot"`
   - `content` = formatted markdown from step 2
   - `confidence` = "high" if samples >= 20, "medium" if >= 5, "low" otherwise
   - `evidence_count` = samples_analyzed
   - `metadata_` = `{}`
4. Upsert ContextCatalog entry using `pg_insert(...).on_conflict_do_update(index_elements=["tenant_id", "file_name"], set_={"status": "active"})` pattern from context.py. Values: `file_name="sender-voice"`, `description="User's writing voice profile extracted from email patterns"`, `status="active"`.

**`delete_voice_from_context(db, tenant_id)`:**
1. Soft-delete all sender-voice entries for this tenant: `UPDATE context_entries SET deleted_at=now() WHERE tenant_id=X AND file_name='sender-voice' AND source='email-voice-engine' AND deleted_at IS NULL`
2. No catalog change needed — catalog stays active, new entry will be written after re-extraction.

Imports needed: `UUID` from uuid, `AsyncSession` from sqlalchemy.ext.asyncio, `update` from sqlalchemy, `pg_insert` from sqlalchemy.dialects.postgresql, `datetime/timezone` from datetime, `ContextEntry/ContextCatalog` from flywheel.db.models, `text` from sqlalchemy.

The `_format_voice_content` helper should handle None values gracefully (skip or show "not yet learned" for fields that are None). For `phrases` (which is a list), join with `", "` and wrap each in quotes. For `avg_length` and `avg_sentences`, combine into one line like `Average length: ~{avg_length} words, ~{avg_sentences} sentences`.

Do NOT call `db.commit()` — the caller owns the transaction.
