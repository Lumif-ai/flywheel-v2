  - Background: `var(--card-bg)`, border: `1px solid var(--subtle-border)`, border-radius 12px

    - Skill chip: `<TaskSkillChip>` if `suggested_skill` exists
  - Right section (flex, gap 8px): three icon buttons
    - Confirm: `CheckCircle` icon, hover color `var(--success)`, aria-label "Confirm task"
    - Later: `Clock` icon, hover color `var(--warning)`, aria-label "Save for later"
    - Dismiss: `X` icon, hover color `var(--error)`, aria-label "Dismiss task"
    - Each button: 36px touch target, 28px visual, `var(--secondary-text)` default color
    - onClick handlers call `onConfirm(task)`, `onLater(task)`, `onDismiss(task)` props
  - Exit animations via CSS classes applied on action:
    - Confirm: translateX(100px) + opacity 0, 150ms ease-in
    - Dismiss: translateX(-100px) + opacity 0, 150ms ease-in
    - Later: translateY(20px) + opacity 0, 150ms ease-in
    - After animation completes (150ms setTimeout or onTransitionEnd), remove from DOM
  - `role="listitem"` for accessibility
  - Props: `task: Task`, `onConfirm`, `onLater`, `onDismiss`, `style?: React.CSSProperties` (for stagger delay)
  - Mobile (<768px): stack actions below content, full-width
