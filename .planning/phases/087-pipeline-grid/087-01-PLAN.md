**Create `hooks/usePipelineMutation.ts`** — Optimistic update mutation following research Pattern 2:
- `usePipelineMutation()` returns a mutation that calls `patchPipelineEntry`
- On mutate: cancel outgoing pipeline queries, snapshot cache, optimistically update the matching item in all cached pipeline query results
- On error: revert to snapshot, show `toast.error('Failed to update')`
- On settled: invalidate all pipeline queries
- Import `toast` from `sonner`, `useQueryClient` from `@tanstack/react-query`

**Create `hooks/usePipelineCreate.ts`** — Create mutation:
- `usePipelineCreate()` returns a mutation calling `createPipelineEntry`
- On success: if `dedup_matched` is true, show `toast.info('Found existing entry: ${name}')`. If false, show `toast.success('Created: ${name}')`.
- On error: `toast.error('Failed to create entry')`
- On settled: invalidate pipeline queries
