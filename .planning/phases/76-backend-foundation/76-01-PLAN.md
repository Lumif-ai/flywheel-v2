4. Return 404 with detail message if skill not found, disabled, or tenant-restricted
5. Return `{"skill_name": skill.name, "system_prompt": skill.system_prompt}` on success (system_prompt may be null -- that is valid)

Place the endpoint AFTER the existing `list_skills` endpoint and BEFORE the run-related endpoints. Add it to the module docstring endpoint list.
