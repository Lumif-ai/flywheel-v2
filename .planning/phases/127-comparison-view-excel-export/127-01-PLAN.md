  <action>
Create the `comparison/` directory and build all utility functions and sub-components.

**comparison-utils.ts** -- Pure functions, no React:
- `formatCurrency(value: number | null, currency: string): string` -- uses `Intl.NumberFormat('en-US', { style: 'currency', currency })`, returns '--' for null. Match pattern from existing `ComparisonMatrix.tsx`.
- `hasMeaningfulDifference(coverage: ComparisonCoverage, threshold = 0.15): boolean` -- returns true if fewer than 2 quotes, if min is 0, or if `(max - min) / min > threshold`. Also returns true if any quote has `has_critical_exclusion` and another doesn't (exclusion difference). Also returns true if any limit differs from required_limit (insufficient difference).
- `getUniqueCarriers(coverages: ComparisonCoverage[]): { name: string; carrier_config_id: string | null }[]` -- deduplicate carriers across all quotes, return in stable order (alphabetical by name).
- `computeCarrierTotals(coverages: ComparisonCoverage[], carriers: { name: string }[]): Map<string, number>` -- sum premiums per carrier across all coverages (skip nulls).
- `isInsufficientLimit(limit: number | null, required: number | null): boolean` -- returns true if both are non-null and limit < required.
- `INSURANCE_CATEGORIES` and `SURETY_CATEGORIES` constants (same values as GapAnalysis.tsx: `['liability', 'property', 'auto', 'workers_comp', 'specialty']` and `['surety']`).

**ComparisonCell.tsx** -- `ComparisonCell({ cell, requiredLimit, currency, highlightBest }: Props)`:
- If `cell` is undefined (carrier didn't quote this coverage): render `<td>` with muted "No quote" text, light gray bg.
- If `cell.has_critical_exclusion`: render `<td className="bg-red-50 border-l-4 border-red-500">` with "EXCLUDED" in `text-red-700 font-semibold text-sm` and `cell.critical_exclusion_detail` truncated in `text-xs text-red-600` with title tooltip.
- If `isInsufficientLimit(cell.limit_amount, requiredLimit)`: render with `bg-amber-50 border-l-4 border-amber-400`.
- Default: render premium in `text-sm font-semibold` (line 1), limit/deductible in `text-xs text-muted-foreground` (line 2). Format: "$X / $Y ded."
- If `highlightBest && cell.is_best_price`: add `text-green-700 bg-green-50` to the cell.
- All cells: `px-3 py-2 border-b border-r whitespace-nowrap`.

**CarrierColumnHeader.tsx** -- `CarrierColumnHeader({ carrier, checked, onToggle }: Props)`:
- Renders a `<th>` with `position: sticky; top: 0; z-index: 20; bg-white`.
- Content: native `<input type="checkbox">` styled with Tailwind (`h-4 w-4 rounded border-gray-300 accent-blue-600`), carrier name below in `text-sm font-medium truncate max-w-[140px]`.
- `onToggle` fires when checkbox changes with `carrier.name`.
- Apply `px-3 py-2 border-b border-r text-center` to th.

**TotalPremiumRow.tsx** -- `TotalPremiumRow({ carriers, totals, currency, selectedCarriers }: Props)`:
- Renders a `<tfoot><tr>` with each `<td>` having `position: sticky; bottom: 0; z-index: 20; bg-white; border-t-2 border-gray-200`.
- First cell: `<td>` with "Total Premium" label, sticky left: 0, z-index: 30 (corner).
- Per-carrier cells: show formatted total from `totals` Map. Bold, slightly larger text (`text-sm font-bold`).
- If carrier is in `selectedCarriers` set, add subtle blue-50 background.

**CriticalExclusionAlert.tsx** -- `CriticalExclusionAlert({ coverages }: { coverages: ComparisonCoverage[] })`:
- Filter coverages to those where any quote has `has_critical_exclusion === true`.
- If none: return null (render nothing).
- Render a banner: `bg-red-50 border border-red-200 rounded-lg p-4 mb-4`.
- Icon: `AlertTriangle` from lucide-react in `text-red-600`.
- Title: "Critical Exclusions Detected" in `text-red-800 font-semibold`.
- List each coverage type with its exclusion detail in `text-sm text-red-700`.
