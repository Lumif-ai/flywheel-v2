**ComparisonView.tsx** -- Main orchestrator component.

Props:
```typescript
interface ComparisonViewProps {
  data: ComparisonMatrix
  currency: string
}
```

State (all local useState, no global store -- per research decision):
- `selectedCarriers: Set<string>` -- initialized with ALL carrier names from `getUniqueCarriers(data.coverages)`. Use `useMemo` for initial value, `useState` initialized lazily.
- `showDifferencesOnly: boolean` -- default `true` (per locked decision).
- `highlightBest: boolean` -- default `false` (per locked decision).

Carrier filtering logic:
- Compute `filteredCoverages` from `data.coverages` by filtering each coverage's `quotes` array to only include quotes where `carrier_name` is in `selectedCarriers`. Use `useMemo`.
- Do NOT remove rows, only filter quotes within rows. A coverage with zero matching quotes after filtering still shows with empty cells.

`onToggleCarrier(name)`:
- If name is in selectedCarriers, remove it. If removing would leave 0 carriers, do nothing (at least 1 must remain).
- Otherwise add it.

Layout (top to bottom):
1. `<CriticalExclusionAlert coverages={data.coverages} />` -- uses FULL coverages, not filtered.
2. `<ComparisonToolbar>` with toggle state and handlers.
3. `<ComparisonTabs>` with filteredCoverages, currency, selectedCarriers, onToggleCarrier, toggle states.

If `data.coverages.length === 0`: show empty state -- centered icon + "No quotes to compare yet. Add carriers and upload quotes to see the comparison matrix." in muted text. Use `BarChart3` icon from lucide-react.
