1. Create `backend/src/flywheel/services/retirement_scanner.py` with an async `retirement_scanner_loop()` function following the exact pattern in `anonymous_cleanup.py` (async loop with try/except, `get_session_factory()`, `asyncio.sleep`). The scanner runs every 3600 seconds (1 hour) and performs four operations in a single transaction:

   a. **Flag stale**: UPDATE pipeline_entries SET stale_notified_at = now WHERE last_activity_at < NOW() - 60 days AND stale_notified_at IS NULL AND retired_at IS NULL AND last_activity_at IS NOT NULL
   b. **Flag no-activity stale**: Same but for entries with last_activity_at IS NULL and created_at < NOW() - 60 days
   c. **Clear stale**: UPDATE pipeline_entries SET stale_notified_at = NULL WHERE stale_notified_at IS NOT NULL AND retired_at IS NULL AND last_activity_at > stale_notified_at (activity happened after stale flag)
   d. **Retire**: UPDATE pipeline_entries SET retired_at = now WHERE stale_notified_at < NOW() - 30 days AND retired_at IS NULL

   Use SQLAlchemy `update()` with `text()` for interval expressions (same pattern as pipeline_service.py stale view filter). Log only when rowcount > 0. No tenant scoping -- this is a system-level maintenance task.

   Constants: SCAN_INTERVAL = 3600, STALE_DAYS = 60, RETIRE_DAYS_AFTER_STALE = 30.

2. Add two methods to `PipelineService` in `pipeline_service.py`:

   `async def retire_entry(self, entry_id: UUID) -> PipelineEntry`:
   - Call `self._get_entry_or_404(entry_id)` (existing helper)
   - Set `entry.retired_at = datetime.now(UTC)`
   - Create an Activity with type="retirement", status="completed", subject="Entry retired"
   - Commit and return entry

   `async def reactivate_entry(self, entry_id: UUID) -> PipelineEntry`:
   - Call `self._get_entry_or_404(entry_id)`
   - Set `entry.retired_at = None` AND `entry.stale_notified_at = None` (both must be cleared)
   - Create an Activity with type="reactivation", status="completed", subject="Entry reactivated"
   - Commit and return entry

3. Register `retirement_scanner_loop` in `main.py` lifespan alongside existing background tasks. Add a `retirement_task` variable, create_task it after the existing tasks, and cancel it in the shutdown block following the same pattern as `cleanup_anon_task`.
