  <files>
    frontend/src/features/settings/components/GranolaSettings.tsx

  <action>
Create GranolaSettings component following the ApiKeyManager.tsx pattern:

**State and queries:**
- `useQuery` to fetch integrations from `GET /integrations/` — filter client-side: `integrations?.find(i => i.provider === 'granola' && i.status === 'connected')` to get `granolaIntegration`. CRITICAL: check BOTH provider AND status — a disconnected integration row may exist.
- `apiKey` controlled input state (string)
- `showKey` toggle for password visibility (Eye/EyeOff icons)

**Mutations:**
- `connectMutation` — POST `/integrations/granola/connect` with `{ api_key: apiKey }`. On success: invalidate ['integrations'], toast.success("Granola connected"), clear apiKey input. On error: toast.error with message.
- `disconnectMutation` — DELETE `/integrations/{id}` using granolaIntegration.id. On success: invalidate ['integrations'], toast.success("Granola disconnected"). On error: toast.error.
- `syncMutation` — POST `/meetings/sync`. On success: invalidate ['meetings'] queryKey, toast.success with sync stats (synced, skipped, already_seen). On error: toast.error.

**Render states:**
1. **Not connected** (no granolaIntegration):
   - Heading: "Granola" with CalendarDays icon
   - Description: "Connect your Granola account to automatically sync meeting notes and transcripts."
   - API key input field with show/hide toggle (password type by default)
   - "Connect" button — disabled when apiKey is empty or connectMutation.isPending
   - Loading spinner on button while connecting

2. **Connected** (granolaIntegration exists):
   - Heading: "Granola" with green CheckCircle2 icon
   - "Connected" status text with green color
   - Last synced: format granolaIntegration.last_synced_at as relative time (or "Never" if null)
   - "Sync Now" button — triggers syncMutation, shows Loader2 animate-spin while pending
   - "Disconnect" button — secondary/destructive variant, triggers disconnectMutation

**Styling:** Use the same card/section styling as ApiKeyManager — rounded border, padding, consistent spacing. Use design tokens from @/lib/design-tokens for colors.
  </action>
