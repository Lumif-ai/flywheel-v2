  <name>Task 1: Granola adapter service</name>
  <files>backend/src/flywheel/services/granola_adapter.py</files>
  <action>
Create new file `backend/src/flywheel/services/granola_adapter.py` implementing three functions. CRITICAL: Use the REAL Granola API shape from 60-RESEARCH.md, NOT the spec's assumed shape.

**Constants:**
- `GRANOLA_API_BASE = "https://public-api.granola.ai/v1"` (NOT `https://api.granola.ai/v1` as spec assumes)

**Dataclasses:**
- `RawMeeting`: external_id (str), title (str), meeting_date (datetime), duration_mins (int | None), attendees (list[dict]), ai_summary (str | None)
- `MeetingContent`: external_id (str), transcript (str), ai_summary (str | None), attendees (list[dict]), metadata (dict)

**Functions:**

1. `async def test_connection(api_key: str) -> tuple[bool, str | None]`:
   - Call `GET /v1/notes?page_size=1` (NOT `/v1/me` which does not exist)
   - 200 -> (True, None); 401 -> (False, "Invalid API key..."); other -> (False, f"Granola API returned {status}")
   - Catch `httpx.RequestError` -> (False, f"Could not reach Granola API: {e}")
   - Timeout: 10s

2. `async def list_meetings(api_key: str, since: datetime | None = None, limit: int = 30) -> list[RawMeeting]`:
   - Call `GET /v1/notes` with params: `page_size=limit`, optionally `created_after=since.isoformat()` (NOT `since` param)
   - Response JSON has key `notes` (NOT `meetings`)
   - Map each item: `item["id"]` -> external_id, `item["created_at"]` -> meeting_date, `item.get("title") or "Untitled"` -> title
   - Compute duration_mins from `calendar_event.start_time` and `calendar_event.end_time` if both present; None otherwise
   - Extract attendees from `calendar_event.invitees` as `[{email, name, is_external: True}]`; default to empty list if no calendar_event
   - `item.get("summary_text")` -> ai_summary (NOT `ai_summary` field)
   - Timeout: 30s

3. `async def get_meeting_content(api_key: str, external_id: str) -> MeetingContent`:
   - Call `GET /v1/notes/{external_id}?include=transcript` (single call, NOT two separate calls as spec assumes)
   - transcript is array of `{speaker, text, start_time, end_time}` -- join as `"[{speaker}]: {text}\n"` lines
   - Extract attendees from `calendar_event.invitees` same as list_meetings
   - Timeout: 60s

Use `httpx.AsyncClient` with context manager for all calls. Set `Authorization: Bearer {api_key}` header.
