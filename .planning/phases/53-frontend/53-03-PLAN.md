- `PipelineItem`: `{ id: string, name: string, domain: string | null, fit_score: number | null, fit_tier: string | null, status: string, last_interaction_at: string | null, outreach_count: number, last_outreach_status: string | null, days_since_last_outreach: number | null, created_at: string }`

- `fetchPipeline(params: PipelineParams): Promise<PipelineResponse>` — `api.get<PipelineResponse>('/pipeline/', { params })`

**hooks/usePipeline.ts**:
- `usePipeline(params: PipelineParams)` — `useQuery(['pipeline', params], () => fetchPipeline(params), { keepPreviousData: true })`

**hooks/useGraduate.ts**:
- `useGraduate()` — `useMutation` calling `graduateAccount`, on success: invalidate both `['pipeline']` and `['accounts']` query keys, show success toast via sonner ("Account graduated to Engaged")

- Columns: Company, Fit Score, Outreach Status, Last Outreach, Days Since Action, Actions

  - **Actions**: "Graduate" button (small, outline variant). On click, calls `useGraduate().mutate(id)`. Shows loading spinner while mutating. After success, row disappears from list (React Query refetch handles this).
