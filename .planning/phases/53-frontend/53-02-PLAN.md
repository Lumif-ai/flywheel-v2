- `fetchAccountDetail(id: string): Promise<AccountDetail>` — calls `api.get<AccountDetail>('/accounts/' + id)`
- `fetchTimeline(accountId: string, params: { offset?: number, limit?: number }): Promise<TimelineResponse>` — calls `api.get<TimelineResponse>('/accounts/' + accountId + '/timeline', { params })`

**hooks/useAccountDetail.ts**:
- `useAccountDetail(id: string)` — `useQuery(['account', id], () => fetchAccountDetail(id), { enabled: !!id })`

**hooks/useTimeline.ts**:
- `useTimeline(accountId: string, params)` — `useQuery(['timeline', accountId, params], () => fetchTimeline(accountId, params), { enabled: !!accountId, keepPreviousData: true })`
