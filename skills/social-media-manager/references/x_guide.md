# X (Twitter) Platform Guide

## Algorithm & Formatting

### What Gets Reach
- Replies and quote tweets drive more reach than standalone posts
- First tweet in a thread is the hook — must work standalone
- Images/videos boost engagement but aren't required
- Posting frequency matters more on X than LinkedIn (3-5/day is fine)
- The LAST tweet in a thread matters too (CTA, follow prompt)

### Character Limits
- Single tweet: 280 characters (free) / 25,000 (Premium)
- Links consume 23 characters regardless of actual URL length
- Thread: unlimited tweets, but 5-8 is the sweet spot
- Alt text on images: 1000 characters

### Tweet Formatting
- No markdown, no formatting — plain text only
- Line breaks work but use sparingly
- Hashtags: 1-2 max, at end of tweet, not inline
- @ mentions: only if the person would genuinely engage

### Thread Structure

```
Tweet 1: [HOOK — must work standalone, stop the scroll]

Tweet 2-N: [BODY — one idea per tweet, each should make sense alone]

Last tweet: [CTA — "Follow for more [topic]" or genuine question]
```

### Hook Patterns for X
- Hot take: "Unpopular opinion: [contrarian view]"
- "I [did thing]. Here's what happened:"
- Thread announcement: "I spent [time] doing [thing]. A thread on what I learned:"
- Challenge: "[Common belief] is wrong. Here's why:"
- Statistic: "[Surprising number] — and nobody's talking about it"

## Content Types

| Type | Best For | Format |
|------|----------|--------|
| Hot take | Engagement, followers | Single tweet |
| Thread | Deep insight, storytelling | 5-8 tweets |
| Quote tweet | Reaction, positioning | Tweet + your take |
| Reply to trending | Visibility, network growth | Reply to relevant post |
| Poll | Engagement, market research | Tweet with poll |
| "TIL" moment | Developer audience | Single tweet |
| Code snippet | Technical credibility | Tweet + screenshot |
| Meme/humor | Virality | Tweet + image |

## Quote Tweets

Powerful for positioning — reacting to others' posts with your lens:
- Add genuine value, not just "Great post!"
- Frame through your role lens: "As someone building compliance tools, this is exactly right because..."
- Contrarian QTs get more reach but higher risk
- Always ask: would the original poster appreciate this reaction?

## X vs. LinkedIn Voice Differences

| Dimension | LinkedIn | X |
|-----------|----------|---|
| Length | 1200-1800 chars | 280 chars (or short threads) |
| Tone | Professional-casual | Casual-direct |
| Frequency | 2-3x/week | 3-5x/day |
| Formatting | Line breaks, structure | Tight, punchy |
| Hashtags | 1-2 | 0-1 |
| Links | In comments | In tweet (less penalty than LinkedIn) |
| Humor | Mild, safe | Appreciated, even edgy |
| Self-promotion | Subtle, story-wrapped | More accepted, direct is fine |

## Posting Schedule

- Volume matters more than timing on X
- Early morning (7-8am) and evening (6-8pm) tend to perform well
- Tech/founder audience is active during work hours
- Weekends can work well (less competition)
- Reply to others during peak hours for visibility
